# QuickVolaInstall

Bash script for a quick Volatility installation

# No volatility 

Just run the script, it itself will do all for you

```bash
sh QuickInstall.sh
```

# With volatility repo

Place this script inside the Volatitlity folder and run it

```
sh QuickInstall.sh
```
